Recently I've been working on a new project which uses a Python server and an Objective-C client. I was used to the easy testing of the REST-like interface on the server and wanted to replicate this for testing on the client in Objective-C.

Unfortunately Objective-C tends to rely on a lot of delegate calls, which can make this sort of testing rather difficult. For this I wrote a simple DelegateTester class which can be included with your test suite to make things easier. An example of usage is:

https://gist.github.com/1730069

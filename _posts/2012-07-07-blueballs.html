After my team fell apart on the planned entry Entropy, I had to scale back quite significantly and decided that the original idea was unworkable.

Instead, I've decided to work on a puzzle game based around blue ball machines. That game is named Blue Balls Inc. and is being written in Javascript with jQuery on the client side and Python using Bottle on the server side.

You can play the latest version of the game at blueballs.jscott.me.
